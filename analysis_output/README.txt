COMO LER OS RESULTADOS
==============================

1. final_report.txt - Resumo executivo com principais conclusões
2. comparison_complete.csv - Dados completos para análise posterior
3. heatmap_model_loss.png - Visualização comparativa de Modelo x Função de Perda
4. ranking_geral.png - Ranking de todos os modelos por Macro F1-Score
5. f1_problematic_classes.png - Desempenho F1-Score em classes problemáticas
6. effectiveness_loss_function.png - Efetividade das Funções de Perda

MÉTRICAS PRINCIPAIS:
- Macro F1: Média harmônica de precision e recall (0-1, maior é melhor)
- Avg Precision: Área sob curva precision-recall (0-1, maior é melhor)
- Hamming Loss: Taxa de erro por classe (0-1, menor é melhor)
